API Key Setup

This app needs your own Google Gemini API key to extract, embed and answer questions. Keys are stored locally in your .env file and never leave your machine.

  1. Go to Google AI Studio → Get API Key
  2. Sign in with a Google account and click Create API key
  3. Copy the key (starts with AIza...) and paste below
  4. Optional: add up to 5 keys to raise the rate limit (parallel rotation)
Saved to .env on the server (chmod 600).

Privacy: Your API keys never leave your server. This app stores them in a local .env file readable only by the owner. To remove a key, clear its field and Save.

Pricing: Gemini has a free tier (see ai.google.dev/pricing). For heavy use, consider a paid plan or rotating multiple keys.
